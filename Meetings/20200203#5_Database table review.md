## Database 필요성?

- 클라이언트에 있어야 하는게 아니라면 결국 서버가 필요
- 서버는 DB의 데이터 말고도 알라딘 API를 통해 데이터를 얻을 수 있으므로
- 클라이언트가 서버에 요청해 데이터를 가져오게 하고, 서버가 판단해서 알라딘 API 호출 or DB Select 하기를 선택적으로 진행
